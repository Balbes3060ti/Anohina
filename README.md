# Как редактировать базу данных сайта

Информация о курсах и обратных звонках редактируется на панели менеджера в личном кабинете администратора. Остальное - в базе данных сайта.

Чтобы войти в базу данных сайта, откройте зайдите в open server panel. У вас появится значок приложения open server panel в трее. Нажмите на него правой кнопкой мыши. Откроется список, в котором вы выбираете "phpmyadmin" -> "Открыть в браузере". Появится страница, где вы нажимаете на ссылку "public". А затем форма входа в phpmyadmin, где вы ничего не меняете, просто нажмите на кнопку "Авторизация".

Поздравляем, вы вошли в phpmyadmin. Здесь находится база данных сайта. Окно phpmyadmin поделено на 2 части. Слева структура базы данных, справа описание и свойства элемента этой структуры. Элементы "lab1_n1_db" и "site_db" - это 2 базы данных, на которые поделена база данных сайта. Нажмите на один из них, и вы увидите её дочерние элементы. Это таблицы со всей информацией о сайте. А как их редактировать, спросите gpt.
